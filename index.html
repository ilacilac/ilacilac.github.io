헥소(Hexo)를 사용하여 gihub.io 블로그 만들기

2020/11/30

헥소(Hexo)란👀?

헥소 공식페이지에 설명되어있기를,

빠르고 간단하고 강력한 블로그 프래임워크라고 설명되어있습니다.

그리고 플러그인과 다양한 테마를 지원하므로

마음에 드는 테마를 다운받아 쉽게 적용하여 사용할 수 있습니다.

설치환경

  • Node.js (Should be at least Node.js 10.13, recommends 12.0 or higher)

  • Git

설치

npm을 통해 전역으로 설치해줍니다.

원하는 디렉토리명으로 hexo init을 한 후,

해당 디렉토리로 이동하여 노드 모듈을 설치해줍니다.

설치 후 터미널에서 hexo server명령어를 실행해 줍니다.

로컬 서버가 구동되면 http://localhost:4000 주소로 들어갑니다.

Hello world로 시작한 포스트 글이 보이면 성공입니다😊!

  2. 원본소스가 저장되어있는 저장소
