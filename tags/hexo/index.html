헥소(Hexo)를 사용하여 gihub.io 블로그 만들기

Nov 30, 2020

헥소(Hexo)란👀?

헥소 공식페이지에 설명되어있기를,
빠르고 간단하고 강력한 블로그 프래임워크라고 설명되어있습니다.

그리고 플러그인과 다양한 테마를 지원하므로
마음에 드는 테마를 다운받아 쉽게 적용하여 사용할 수 있습니다.

설치환경

  • Node.js (Should be at least Node.js 10.13, recommends 12.0 or higher)
  • Git

설치

npm을 통해 전역으로 설치해줍니다.

$ hexo init hexo
$ cd hexo
$ npm install

deploy:
type: ‘’

deploy:
type: git
repo: 저장소 주소(예: https://github.com/사용자명/사용자명.github.io.git)
branch: master

npm install –save hexo-deployer-git

$ hexo generate

$ hexo deploy

$ hexo clean
